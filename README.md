# LifeTrack — 个人时间管理与追踪

一个纯本地存储的个人效率管理应用。无需登录、无需服务器，数据完全保存在设备本地。

> 本项目由「家教排课软件」转型而来，从面向机构的多端同步工具，简化为专注个人使用的本地化应用。

---

## 项目结构

```
.
├── lifetrack-web/           ← Web App（推荐主力使用）
│   ├── src/
│   │   ├── db.ts            # Dexie.js 本地数据库
│   │   ├── App.tsx          # 主应用
│   │   ├── components/      # 页面组件
│   │   ├── test/            # 测试套件（224+ tests）
│   │   └── utils/           # 工具函数（CSV 导出等）
│   └── README.md
│
└── tutor-schedule/          ← Android App（原生体验备选）
```

## 快速开始

```bash
cd lifetrack-web
npm install
npm run dev
```

浏览器访问 `http://localhost:5173/`

---

## 核心交互

| 操作 | 方式 |
|------|------|
| 切换页面 | 底部导航栏 或 **Ctrl+1~8** 快捷键 |
| 全局搜索 | **Ctrl+K** 快速定位 |
| 把任务排进课程表 | 课程表页 → 展开「任务」面板 → **拖拽** 任务到日程格子 |
| 调整课程时间 | **拖拽** 课程卡片到新的日期/时间段 |
| 标记任务完成 | 任务页点击状态圆环 / Dashboard 快捷操作 |
| 批量操作 | 任务页点击「批量」→ 多选 → 标记完成 / 分配目标 / 删除 |
| 数据备份 | 设置页 → 导出 JSON / 导出 CSV |
| 离线使用 | PWA 添加到主屏幕后，无网络也能正常使用 |

## 数据说明

- **不上传任何数据到服务器**
- **无需注册/登录**
- 数据保存在浏览器 IndexedDB 中
- 支持 JSON 导出/导入备份
- 支持 CSV 导出用于 Excel 分析
