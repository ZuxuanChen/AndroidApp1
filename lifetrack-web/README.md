# LifeTrack — 个人时间管理与追踪

一个纯前端、纯本地存储的个人效率管理 Web App。无需登录、无需服务器，打开浏览器即可使用，数据保存在本地 IndexedDB 中。

---

## 功能概览

### 📊 总览 Dashboard
打开 App 的第一屏，一屏掌握全天信息：
- **今日工作量** — 统计今天完成的任务及累计工作时长
- **今日日程** — 当天的课程/安排一览
- **重要任务** — 待办的高优先级任务
- **目标进度** — 各目标的完成比例（进度条）
- **近期睡眠** — 最近几天的睡眠时长与质量

### 📅 课程表
周视图排课，支持交互式操作：
- **点击添加/编辑课程** — 设置名称、时间、时长、颜色、地点
- **拖拽调整位置** — 按住课程卡片拖到其他日期或时间段，自动按 30 分钟刻度对齐
- **从任务拖拽创建课程** — 展开任务面板，将任务拖入课程表，自动关联任务
- **一键完成** — 关联了任务的课程卡片右上角显示「✓」按钮，点击即可标记任务完成
- **已完成划掉** — 关联任务已完成的课程自动显示删除线 + 降低透明度

### ✅ 任务追踪
- **状态管理** — 待办 / 进行中 / 已完成，点击圆环快速切换
- **优先级** — 高 / 中 / 低
- **单次 / 多次** — 单次任务安排到课程表后自动从面板隐藏；多次任务可反复安排
- **关联目标** — 任务可绑定到具体目标，便于追踪

### 🎯 目标管理
- 设定长期/短期目标，带截止日期
- 每个目标分配独立颜色
- 自动统计该目标下的任务完成比例

### 🌙 睡眠记录
- 记录入睡/起床时间
- 1-5 星睡眠质量评分
- 自动计算睡眠时长
- 统计平均睡眠与平均质量

## 技术栈

| 技术 | 说明 |
|------|------|
| React 19 + TypeScript | UI 框架 |
| Vite | 构建工具 |
| Tailwind CSS | 样式 |
| Dexie.js | IndexedDB 封装，本地数据存储 |
| Lucide React | 图标库 |
| PWA | Service Worker + Manifest，支持离线使用和添加到主屏幕 |

## 快速开始

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 浏览器访问 http://localhost:5173
```

### 构建生产版本

```bash
npm run build
```

构建产物在 `dist/` 目录，可部署到任何静态托管服务（Vercel、Netlify、GitHub Pages 等）。

## 数据说明

所有数据保存在浏览器 **IndexedDB** 中，完全本地：
- 不上传任何数据到服务器
- 无需注册/登录
- 换设备或清除浏览器数据会丢失（后续可添加 JSON 导出/导入功能）

---

## 核心交互设计

| 场景 | 操作方式 |
|------|---------|
| 把任务排进课程表 | 课程表页 → 展开「任务」面板 → **拖拽**任务到日程格子，或点击「安排」按钮 |
| 调整课程时间 | **拖拽**课程卡片到新的日期/时间段 |
| 标记任务完成 | 课程表上点击课程卡片的「✓」按钮，或任务页点击圆环 |
| 单次 vs 多次任务 | 创建任务时选择「单次」（安排后从面板隐藏）或「多次」（可反复安排） |

## 部署到线上

```bash
npm run build
```

将 `dist/` 目录内容上传到：
- [Vercel](https://vercel.com)（推荐）
- [Netlify](https://netlify.com)
- [GitHub Pages](https://pages.github.com)
- 任何静态文件托管服务

手机访问后，在浏览器菜单中选择「添加到主屏幕」，即可像原生 App 一样使用。
